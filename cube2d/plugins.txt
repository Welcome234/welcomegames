DiscordChat | This plugin adds Discord integration to your server. Chat messages will be bridged between the Cube2D server chat and Discord.[newline][newline]How to setup this plugin:[newline]- Create an application in discord developer portal[newline]- In the installation tab, disable User install and set install link to none[newline]- In the Bot tab, enable Presence Intent, Server Members Intent and Message Content Intent[newline]- In the Bot tab, click reset token[newline]- Copy the bot token into DiscordChat config.txt[newline]- In General Information, copy the application ID[newline]- In your browser, go the the following URL https://discordapp.com/oauth2/authorize?scope=bot+applications.commands&client_id=(paste your application ID here)[newline]- Select your server and click Authorize[newline]- Copy the discord channel ID you want to link and paste in DiscordChat config.txt[newline]- Ensure your bot has administrator permission[newline][newline]This plugin will work on any version from 1.13.1.
